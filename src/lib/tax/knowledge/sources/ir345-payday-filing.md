# Payday Filing

All employers must file employment information with IRD electronically within **2 working days** of each payday (payday filing).

## What to file
Each filing includes:
- Employee's IRD number, name, and tax code
- Gross earnings and PAYE deducted
- KiwiSaver deductions (employee and employer)
- ESCT on employer KiwiSaver contributions
- Student loan deductions
- Child support deductions

## How to file
- Through payroll software (most common)
- Through myIR (IRD's online portal)
- File upload in CSV or electronic format

## Schedular payments (contractors)
If you pay contractors for schedular activities, you may need to withhold tax:
- Contractor provides an IR330C form with their elected withholding rate
- If no form provided, withhold at the **no-notification rate** (45%, or 20% for non-resident contractors)
- File contractor payment information similarly to employment information
- Common schedular activities: IT contractors, labourers, consultants, entertainers
